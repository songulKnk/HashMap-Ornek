# HashMap-Ornek
HashMap ile öğrenci ve okul no eşleştirilip noya göre öğrenci bulunur
Ogrenci classında kullanacağımız değişkenler ve bunların setter getterlarını oluşturdum
Test classında ise bir hashmap oluşturup elle bu hashmape veriler ekledim.
(Okul no'ya göre öğrenci) 
Test classını çalıştırın.
(Containskey ile arar) girilen numaraya ait bir öğrenci varsa,
aşağıdaki çıktıyı alırsınız:
Öğrenci nosunu girin : 12
Öğrenci: Ogrenci [isim=Songül, ogrNo=12, sinif=4, cinsiyet=KIZ]

Bir öğrenciye ait olmayan okul no girdiğinizde aşağıdaki çıktıyı alırsınız:
Öğrenci nosunu girin : 444
Bu numaraya sahip bir öğrenci yok
